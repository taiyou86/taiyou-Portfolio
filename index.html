My Skills

HTML/CSS

HTMLでSEOを意識した構造的なマークアップを行い、CSSを使ってデザインカンプに基づいた正確なレイアウトとスタイリングを実現します。

JavaScript

イベントハンドリングからDOM操作、アニメーションの実装まで幅広く対応可能です。WEB制作でよく使う動きの実装は問題なくできます。

PHP

条件分岐や繰り返し処理などの基本的なPHPプログラムから、フォームのバリデーション、データベース操作、SQLとの連携も行います。Laravelフレームワークにも対応可能です。

MySQL

PHPと連携し、MySQLデータベースに対してデータの保存、更新、削除、検索（CRUD操作）をすることが可能です。

git

Gitを使ってバージョン管理を行い、チーム開発における効率的なコラボレーションが可能です。マージ、コンフリクト解消なども問題なくできます。

WordPress

WordPressを用いて、SEOに最適化されたウェブサイトを構築します。テーマカスタマイズやプラグインの導入、モバイル対応も行い、要望に合わせた管理しやすいサイトを作成します。
